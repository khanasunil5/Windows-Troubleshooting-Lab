# Troubleshooting Internal DNS Issues

## Symptoms
- Network connected
- Cannot access domain resources

## Cause
Incorrect DNS configuration on client.

## Fix
1. Verify DNS server using ipconfig /all
2. Update DNS to Domain Controller IP
3. Flush and register DNS cache

## Verification
Hostname resolution and domain access restored.
