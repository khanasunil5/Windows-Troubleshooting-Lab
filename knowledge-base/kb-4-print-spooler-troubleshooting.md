# Printer Spooler Troubleshooting

## Symptoms
- Printer configuration error
- Jobs fail immediately

## Cause
Print Spooler service stopped or blocked.

## Fix
1. Restart Print Spooler
2. Clear spooler cache
3. Reinstall printer
4. Verify printer port

## Verification
Test print successful.
