# NTFS vs Share Permissions

## Symptoms
- Access denied to shared folder

## Cause
Missing NTFS permissions despite share access.

## Fix
1. Review share permissions
2. Review NTFS permissions
3. Use Effective Access to confirm
4. Apply least-privilege NTFS permissions

## Verification
User gains appropriate access.
