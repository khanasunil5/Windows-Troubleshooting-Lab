# Reset Windows Update Components

## Symptoms
- Updates failing or stuck

## Cause
Windows Update or BITS services stopped.

## Fix
1. Restart required services
2. Clear SoftwareDistribution\Download
3. Retry Windows Update

## Verification
Updates install successfully.
